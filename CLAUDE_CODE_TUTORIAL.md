# Claude Code 入门教程

## 什么是 Claude Code？

Claude Code 是 Anthropic 官方推出的命令行 AI 编程助手，直接在终端中运行。它能理解你的代码库、编辑文件、执行命令、回答代码问题，让开发效率翻倍。

## 安装与启动

### 安装
```bash
npm install -g @anthropic-ai/claude-code
```

### 启动
在项目目录下运行：
```bash
claude
# 或简写
cl
```

### 首次使用
首次启动会提示登录 Anthropic 账号，按指引完成即可。

## 核心操作模式

### 对话模式（默认）
直接输入问题或需求，Claude 会理解并执行：
```
你：帮我给这个函数加个单元测试
```

### 斜杠命令
| 命令 | 作用 |
|------|------|
| `/help` | 查看所有命令 |
| `/clear` | 清空对话 |
| `/init` | 生成 CLAUDE.md 项目指南 |
| `/review` | 审查代码变更 |
| `/plan` | 制定实现计划 |
| `/memory` | 查看/编辑持久记忆 |
| `/context` | 查看上下文使用情况 |
| `/config` | 修改设置 |
| `/model` | 切换模型 |
| `/loop` | 定时循环执行 |

## 常用工作流

### 1. 理解项目
进入项目后先说：
```
先看一下这个项目的结构
```

### 2. 修改代码
```
在 src/utils/date.js 里加一个格式化日期函数
```

### 3. 修复 Bug
```
登录页面点击按钮没反应，排查一下
```

### 4. 代码审查
```
/review
```

## 核心概念

### CLAUDE.md
项目根目录的 `CLAUDE.md` 文件，告诉 Claude 项目的结构、启动方式、编码规范。用 `/init` 自动生成。

### 技能（Skills）
自定义斜杠命令，放在 `.claude/skills/<名称>/SKILL.md` 中：
```markdown
---
name: deploy
description: 部署到生产环境
---
1. 运行测试
2. 构建项目
3. 推送到主分支
```

### 持久记忆
Claude 可以记住用户偏好、项目上下文。用 `/memory` 管理。

### MCP 工具
扩展 Claude 能力的插件系统（数据库查询、浏览器控制、API 调用等）。

## 权限模式

Claude Code 有三级权限：

| 模式 | 说明 |
|------|------|
| **自动允许** | 常见安全操作自动执行 |
| **每次询问** | 高风险操作（删除文件、推送代码）需确认 |
| **拒绝** | 用户明确禁止的操作 |

## 实用技巧

### 高效提问
- **给上下文**：在提需求前先让 Claude 了解相关代码
- **分步进行**：复杂任务拆成多个小步骤
- **明确反馈**：不满意时说"换个方式"而不是"不对"

### 文件管理
- Claude 会自动跟踪你修改的文件
- 用 `git diff` 查看变更
- 确认后再提交

### 上下文管理
- `/context` 查看剩余上下文
- 长对话后可用 `/compact` 压缩
- 复杂任务尽量在一个会话中完成

### 快捷键
| 快捷键 | 作用 |
|--------|------|
| `Ctrl+C` | 停止当前操作 |
| `↑` | 上一条指令 |
| `Ctrl+L` | 清屏 |

## 常见问题

**Q：Claude Code 会修改我的代码吗？**
A：只有你确认后才会执行修改，每次操作前都会先展示变更。

**Q：支持中文吗？**
A：完全支持中文对话和代码注释。

**Q：怎么控制权限？**
A：用 `/config` 调整权限模式，或修改 `settings.json`。

**Q：能在团队中使用吗？**
A：可以，CLAUDE.md 和技能文件可以提交到 Git，团队共享。
